# ZenVoiceAI 總控制台 - 產品需求文件 (PRD)

## 1. 產品概述
**ZenVoiceAI 總控制台** 是一個統一微服務架構的網頁主控台。其主要目的是將分散在各個獨立服務埠 (Ports) 的佛學語音與文字處理工具整合到一個統一、簡潔、且具備「禪意 (Zen)」風格的使用者介面中。讓使用者能夠在單一視窗中無縫切換多種專業工具，並透過 SPA 架構保留所有操作狀態。

## 2. 目標受眾
- 需要處理佛教經典、語音合成、以及學習佛法名詞的專業使用者或法師。
- 需要沉浸式、無干擾閱讀與聆聽環境的使用者。

## 3. 核心功能需求

### 3.1 統一導覽列 (Unified Navigation)
- 提供頂部導覽列，包含五大核心子系統的切換按鈕：
  1. **智能語音MP3播放** (連接至本機 5000 埠)
  2. **Cbeta名詞查詢** (連接至本機 8001 埠)
  3. **文字轉檔語音** (獨立模組)
  4. **發音校正字典** (獨立模組)
  5. **逐行朗讀文字** (連接至本機 5173 埠)
- 點擊按鈕時需有明顯的 Active 狀態與平滑的視覺回饋。

### 3.2 狀態駐留架構 (SPA State Preservation)
- **需求**：使用者在工具（如逐行朗讀或 Cbeta 查詢）中輸入的文字、載入的檔案與播放進度，在切換到其他工具後必須被完整保留。
- **解決方案**：採用「多重視窗駐留」機制。切換工具時不得重新載入網頁或重置 iframe 來源，而是將不在畫面上的工具隱藏至背景運行。

### 3.3 禪意載入體驗 (Zen-style Loader)
- 系統在第一次載入某個子系統時，需展示具備沉浸感與禪意的 Loading 動畫與提示文字（「正在為您準備服務，請稍候...」），直到該系統完全準備就緒。

### 3.4 全域筆記區 (Global Notes)
- 左側保留固定寬度的全域筆記專區。
- 使用者在右側子系統（如 Cbeta 名詞查詢）查閱資料時，可隨時將重點摘錄至左側面板，方便後續匯出。

## 4. 介面設計規範 (UI/UX)
- **風格**：深色模式 (Dark Theme)、禪意、護眼、高對比但低刺眼感。
- **排版**：左側筆記區與操作提示佔據次要空間 (Left Panel)，右側為主功能動態展示區 (Right Section)。
- **動畫**：切換工具時需有淡入淡出的平滑漸變 (Opacity Transition) 效果。
